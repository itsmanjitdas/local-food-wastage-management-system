# 🥗 Local Food Wastage Management System

An end-to-end data analytics and logistics optimization pipeline built to connect surplus food providers (restaurants, grocers) with local receivers (NGOs, shelters) using Python, MySQL, and Streamlit.

## 🚀 Key Features
- **Live KPI Tracking:** Dynamic card arrays measuring overall ecosystem metrics.
- **Automated Data Cleaning:** Robust ingestion pipeline normalizing datetime schema discrepancies via Pandas.
- **Interactive Query Engine:** A dropdown console hosting 15 advanced analytical SQL operations.
- **Full Inventory Control:** Functional CRUD operations interface allowing creation, state updates, and permanent record purge constraints.

## 🛠️ Technical Stack
- **Frontend:** Streamlit, Plotly Express
- **Backend/Logic:** Python 3.x, Pandas
- **Database:** MySQL Server
- **Driver:** mysql-connector-python
